# Martin Mega Bingo 🎉

A dark-themed customizable 3x3 bingo game with support for:

- Text entries
- Images
- Audio clips
- Confetti win animation
- Responsive design

## Features

✅ 3x3 bingo grid  
✅ Fully customizable via JavaScript  
✅ Supports text, images, and audio  
✅ Dark modern UI  
✅ Win detection  
✅ Confetti celebration animation
